# Dotfiles
Por el momento es una mezcla entre dotfiles y [esto](https://gist.github.com/marcorichetta/137de5decf91a2c028bc632f9443da8f)

## 1. `fish.sh`

Instalar fish shell y setearlo como predeterminado.

`chsh -s /usr/bin/fish`

**(Requiere cerrar la sesión)**

## 2. `install.sh`
Este debiera ser el que ejecuta a los demás.

## 3. `programs.sh`
Acá están los comandos para instalar los programas.

## 4. `backup.sh`
Mueve los config files de /backup a sus respectivos directorios.
